# Dissolução de Sais em Água - Exemplo NaCl

Este projeto é uma página educacional interativa e responsiva sobre a dissolução de sais em água, usando o NaCl como exemplo.  
Foi desenvolvido com **HTML5, CSS3 e JavaScript puro**, pronto para publicação no **GitHub Pages**.

## 📂 Estrutura
- `index.html` → Página principal
- `style.css` → Estilos modernos e responsivos
- `script.js` → Anima
